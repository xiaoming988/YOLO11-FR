# Contributing

This repository is primarily provided to support reproducibility of the manuscript:

**YOLO11-FR: A bridge crack detection method based on frequency-domain fusion and an edge enhancement mechanism**

Issues and pull requests are welcome for bug fixes, documentation improvements, and reproducibility-related suggestions.

Before contributing, please make sure that:

1. Your changes do not include private datasets, model weights, or large generated files.
2. Your code is documented clearly enough for reproducibility.
3. Your changes are consistent with the license terms in the `LICENSE` file.
